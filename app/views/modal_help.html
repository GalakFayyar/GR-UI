1. La carte

1.1. Indicateurs de la carte

  • Potentiel d'audience

  • Ne s’active que lorsqu’une activité est sélectionnée. Découpage en 4 tranches (par activité) du nombre de recherches inconnues (ou pures) sur les 12 derniers mois glissants. Le nombre exact de recherches inconnues se trouve dans la pop up localité : « Nb recherches » Exemple d’une recherche inconnue ou activité pure : « Plombiers » à Sélestat ou « Garages Automobiles » à Perpignan.

  • Parution (Option filtres)

  • Ne s’active que lorsqu’un établissement et une activité sont sélectionnés. (SI le bouton ON est activé) Ce sont les produits valides en cours de parution OU les produits validés qui vont paraitre (exemple une pôle position vendue et en parution seulement dans 6 mois). Lorsque vous activez le filtre « parution », il est possible d’afficher les produits du professionnel sélectionné, sur l’activité sélectionnée. En choisissant un produit SEARCH, un contour bleu apparaitra sur les localités que le professionnel a souscrites S’il a souscrit un produit départemental, la liste des départements souscrits apparaitra dans le menu filtre.

  • Les établissements (Option filtres)

  • Ne s’active que lorsqu’une activité est sélectionnée. (Ou une activité + un établissement) Il est possible d’afficher les prospects en noir (rubrique officielle de parution sur l’activité sélectionnée) et/ou les clients en bleu (au moins 1 € TS dépensé sur l’activité sélectionnée).

  • Zone de chalandise (Option filtres)

  • Il est possible de faire une zone de chalandise :

    • autour d’un établissement (si vous avez sélectionné un établissement + une activité)
    • autour du centre d’une localité (si vous avez sélectionnée une activité + une localité)
    Avec le curseur, vous pouvez définir une zone de 0 à 100 km.

  • Zone Urbaine (ZU)

  • En arrière plan, il est possible de voir le contour autour des ZU des principales agglomérations de France. Certaines activités ne sont pas soumises aux ZU. Il y a 160 ZU à date. Pour rappel : un client présent dans la ZU d’une ville peut paraitre localement sur cette grande ville.

1.2. Les pop up

  • Pop up localités
    • Nom de la localité
    • Code Insee
    • Nb recherches pures : Nombre de recherches inconnues (pures) sur les 12 derniers mois glissants
    • Locaux rankés : Nombre de blocs locaux rankés sur l’activité * localité
    • Locaux non rankés : Nombre de blocs locaux non rankés sur l’activité * localité
    • Extralocaux rankés : Nombre de blocs extralocaux rankés sur l’activité * localité
    • Extralocaux non rankés : Nombre de blocs extralocaux non rankés sur l’activité * localité
  • Pop up professionnel
    • Client :
      • Raison Sociale
      • Code EPJ
      • Rubrique principale : L’activité principale du professionnel en terme de ventes TS
      • Montant TS :
      • Montant SEARCH
      • Montant SITES
      • Montant DISPLAY
      • Montant PRINT
    • Prospect :
      • Raison Sociale
      • Code EPJ
      • Rubrique officielle : L’activité principale du professionnel (la rubrique officielle (RO))
      • Appétence prospect : Note de 1 à 4 étoiles (Plus le prospect a d’étoiles plus la probabilité est forte de pouvoir le transformer)

2. Les données

2.1. Potentiel d’audience

Découpage en 4 tranches (par activité) du nombre de recherches inconnues (ou pures) sur les 12 derniers mois glissants. Le nombre exact de recherches inconnues se trouve dans la pop up localité : « Nb recherches pures » Exemple d’une recherche inconnue ou activité pure : « Plombiers » à Sélestat ou « Garages Automobiles » à Perpignan.

2.2. Parution

Ce sont les produits valides en cours de parution OU les produits validés qui vont paraitre (exemple une pôle position vendue et en parution seulement dans 6 mois).

2.3. Montants du plan de parution du professionnel

Ce sont les montants par support du dernier plan de parution traité intégralement sur une édition.

2.4. Constitution des listes réponses

Dernière requête la plus recherchée sur les trois derniers mois produisant une liste réponse contenant l’activité sélectionnée.

Exemple :

Photographe à Sélestat est la requête la plus recherchée qui ramène dans la liste réponse deux activités (Photographe publicitaire et Photographe de portraits). Si vous tapez sur PJ Photographe publicitaire (jamais recherché tel quel), vous n’aurez pas le même résultat pour la LR. Il est donc possible d’avoir deux LR identiques pour deux activités différentes.

2.5. Données d’audience

Pour obtenir toutes les données d’audience, il faut exporter les données du panier en PDF ou en CSV.

3. Filtres

3.1. Zone de chalandise

Si vous avez sélectionné un établissement, il est possible de dessiner une zone de chalandise autour de ce professionnel (entre 0 et 100km). Si vous avez sélectionné uniquement une activité (pas d’établissement sélectionné), il est possible de dessiner une zone de chalandise autour du centre de la localité (entre 0 et 100km)

3.2. Les établissements

Il est possible d’afficher les établissements concurrents sur l’activité sélectionnée (soit tous les établissements, les prospects ou les clients).

3.3. Les parutions

Il est possible d’afficher toutes les parutions en cours ou pas encore en parution d’un professionnel sur l’activité sélectionnée.

4. Panier

4.1. Tableau

Il est possible d’ajouter des données sur la localité * activité au panier : Trois façons

  • Une par une
  • En cliquant sur la pop up d’une localité possédant des informations d’audience et en en cliquant sur « ajouter au panier »
  • Toutes les données de l’écran
  • En cliquant sur le bouton « toutes les données de l’écran » dans le panier. Vous aurez ainsi toutes les données de l’écran qui s’afficheront dans le panier sur l’activité sélectionnée
  • Toutes les données de la zone de chalandise
  • En cliquant sur le bouton « toutes les données de la zone de chalandise » dans le panier (si la zone de chalandise est activée et affichée pour une localité ou pour un professionnel). Vous aurez ainsi toutes les données de la zone de chalandise qui s’afficheront dans le panier sur l’activité sélectionnée.

Si vous ajoutez plusieurs fois la même localité sur une activité donnée, elle n’apparaitra qu’une seule fois dans le panier. Il est possible de supprimer des localités une par une du panier ou de vider le panier totalement Si votre panier est conséquent, il est également possible de filtrer par activité et/ou par localité en tapant dans les barres prévues à cet effet.

4.2. Export

Il est possible d’exporter les données en CSV ou en PDF. Vous aurez à disposition dans le document toutes les données de recherches, clics et contacts sur l’ensemble des activités*localités sélectionnés.

5. Laissez un commentaire

Il est possible que vous laissiez un commentaire si vous avez identifié d’éventuels problèmes ou si vous souhaitez partager sur d’éventuels souhaits d’évolution ou simplement faire un retour sur l’outil (cas concret d’utilisation)
